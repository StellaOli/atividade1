Testes Automatizados com Pytest

## Sobre o Projeto

Este projeto contém implementações e testes automatizados para simulações computacionais. Utilizamos Python e Pytest para realizar a atividade

## Estrutura do Projeto

O projeto está organizado com:
- Módulos Python contendo as implementações das simulações
- Testes unitários usando Pytest para validar as funcionalidades
- Documentação e exemplos de uso

## Tecnologias Utilizadas

- **Python**: Linguagem principal do projeto
- **Pytest**: Framework para testes automatizados

## Configuração do Ambiente Virtual

Para configurar o ambiente virtual para este projeto, siga os passos abaixo:

1. **Criar o Ambiente Virtual**:
   Execute o seguinte comando no terminal:
   ```bash
   python -m venv venv
   ```

2. **Ativar o Ambiente Virtual**:
   - No Windows:
     ```bash
     venv\Scripts\activate
     ```
   - No macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

3. **Instalar Dependências**:
   Após ativar o ambiente virtual, instale as dependências necessárias com o comando:
   ```bash
   pip install -r requirements.txt
   ```
## Como Executar os Testes

Para executar todos os testes:
```bash
pytest
```

Para executar com verbose:
```bash
pytest -v
```

## Objetivo

O objetivo é aplicar conceitos de simulação computacional enquanto mantemos boas práticas de desenvolvimento de software, incluindo testes automatizados e documentação adequada.
